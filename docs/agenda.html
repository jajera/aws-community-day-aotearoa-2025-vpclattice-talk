Session Agenda

VPC Lattice: Modern Service Networking

← Back to Talk

Session Overview

This session will explore AWS VPC Lattice, a fully managed service that simplifies service-to-service communication and networking. We'll cover the fundamentals, use cases, and hands-on implementation.

Duration

30 minutes

Level

Advanced

Learning Outcomes

  • ✓ Understand VPC Lattice concepts and architecture
  • ✓ Learn when and how to use VPC Lattice vs. other networking solutions
  • ✓ Implement a basic VPC Lattice service network
  • ✓ Configure service discovery and load balancing
  • ✓ Apply security best practices for service networking

Prerequisites

  • • Basic understanding of AWS VPCs and networking
  • • Familiarity with microservices architecture
  • • Basic Terraform knowledge (for hands-on portion)
  • • AWS account with appropriate permissions

Session Outline

1. Introduction (5 minutes)

Welcome, speaker introduction, and session overview

2. VPC Lattice Deep Dive (10 minutes)

Core concepts, architecture, and key features

  • • Service networks and service groups
  • • Service discovery and load balancing
  • • Security and access control
  • • Monitoring and observability

3. Demo (5 minutes)

Demonstration of a working VPC Lattice application built with Terraform

5. Q&A and Wrap-up (5 minutes)

Questions, discussion, and next steps
